# my-first-repo

Project Description

The Student Grade Calculator is a simple Java console-based mini project designed for beginners.
It takes marks of 5 subjects as input, calculates the total, average, and assigns a grade based on the average score.
This project helps beginners understand core Java concepts like loops, conditional statements, and user input.

Technologies Used

Java
Scanner Class
Command Line / Console

 Features
 
Accepts marks for 5 subjects
Calculates total marks
Calculates average
Displays grade based on average
Simple and beginner-friendly code

Concepts Covered

Loops (for)
Conditional statements (if-else)
User input (Scanner)
Arithmetic operations
